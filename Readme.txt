Bài học đầu tiên:
git init -- Khởi tạo kho Git local
git status --- xem trạng thái kho. Nếu thấy có file báo mầu đỏ --> untrack (chưa nằm trong kho)
git log  --- xem lại log
tạo tài khoản github --> truy cập vào trang github.com
Có thể tạo tài khoản bằng email
Sau khi có tài khoản xong:
1. Tạo Repository trên github
2. Tạo branch -> nhánh main
3. kết nối đết github thông qua link trên github
4. push
